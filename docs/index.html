🚀 AuditHub

AuditHub is a lightweight open-source CLI tool to audit GitHub repositories and track stale pull requests, missing documentation, and branch clutter.

🔧 Features

  • Detect PRs open > 30 days
  • Check for missing README.md
  • Count branches and repo activity

📦 How to Use

python audit.py --token <your_token> --repo owner/repo

📈 Roadmap

  • GitHub Actions integration
  • Slack & Teams alerts
  • Web dashboard
